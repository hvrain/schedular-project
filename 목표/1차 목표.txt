스케줄러 1차 목표

1. 모든 일과를 한눈에 볼 수 있는 메인 페이지 구성
- 보여지는 계획표의 순서를 커스텀 가능하게 구현
- 사용자 별로 다양한 형태의 페이지 형태를 구성할 수 있도록 구현
  ex) 티스토리의 블로그 테마
2. 계획표를 작성할 수 있는 기능 구현
- 계획의 종류별로 적합한 작성툴을 구현
  ex) 여행 계획표 -> 금전 계획표, 여행 일정 계획표, 이동 계획표
- 월별, 일별, 시간별, 중요도별 등 다양한 계획도 구성
3. 계획표를 공유할 수 있는 기능 구현
- 계획표 공유하기를 누르면 해당 계정을 구독하고 있는 사람들에게 보여주기(일단은 트래픽 고려x)
4. 로그인 기능 구현
- 로그인하면 다른 사용자를 구독하고 좋아요를 누르고 일정을 등록할 수 있음
- 로그인하지 않아도 사용자에게만 계획표가 보여지도록 구현(캐시를 통해)
